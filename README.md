# BlogAngularAsyncTesting

Angular Advent Calendar 2021のサンプルコードです。

## ディレクトリ

サンプルのテストコードは `src/app/tests` に保存されています。

## テストの実行

下記のコマンドでテストを実行できます。

```sh
ng test
```

初回のテスト実行時は下記コマンドも合わせて実行してください。

```sh
# v16.10以上で実行してください
node -v

# パッケージをインストールしてください
npm ci
```

## その他

通らないテストの例は `xit` でスキップしています。
